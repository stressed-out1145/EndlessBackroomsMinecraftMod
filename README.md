**Introduction**

Welcome to Endless Backrooms, a survival horror mod that brings the Backrooms into Minecraft. Start your journey in the iconic Level 0, gather resources, and try to survive the entities lurking in the shadows. Can you find the way out, or will you be lost forever?

**How to start**

Create a new world in Minecraft. Then, find a way to get stuck in blocks. Upon dying from suffocate, you will noclip directly into Level 0. Good luck, Wanderer.
Levels:
- level 0
- level 1 (coming soon)

**引言**

欢迎来到 Endless Backrooms (无尽后室)，这是一款将后室带入 Minecraft 的生存恐怖模组。从标志性的 Level 0 开始你的旅程，收集资源，并试图在潜伏于阴影中的实体手下存活。你能找到出路，还是将永远迷失？

**如何开始**

在 Minecraft 中创建一个新世界。设法让自己窒息（被方块卡住）。当你因窒息而死亡时，你将直接卡入 Level 0。祝你好运，流浪者。
